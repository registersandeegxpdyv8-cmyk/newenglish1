# 新概念英语第一册学习工具

一个离线可用的单页学习工具，用来按课练习新概念英语第一册。

## 功能

- 144 课课程导航
- 奇数课课文精读
- 偶数课隐藏独立课文，只做单词和句型操练
- 单词卡片：英文、音标、词性、中文
- 单词点击发音
- 课文英文朗读：只读英文，并按句子停顿
- 单词测验：英文到中文、中文到英文、音标到英文、词性+中文到英文；生成后自动隐藏单词卡，用左右滑动卡片逐题练习，答对自动跳到下一题
- 跟读练习：像微信聊天一样长按语音条发语音
- 每条跟读语音可点击回放，方便对比自己有没有进步
- 跟读是独立板块，支持任意单词或句子，也可以不输入目标直接让工具猜你读了什么
- 浏览器会识别你读成了什么，并给出纠正建议
- 语法练习：进入语法页先显示本课语法知识点详细讲解；点击生成练习后进入左右滑动题卡，题目会直接给出具体转换或翻译要求，可自动检查常见错误和明显词义不匹配
- 每个语法练习可单独检查或删除
- 翻译句子训练
- 本地保存、导入、导出学习数据

## 使用

直接打开 `index.html`。

发音练习需要浏览器支持 Web Speech Recognition。建议使用 Chrome 或 Edge，并允许麦克风权限。

每个人打开链接后，学习记录都保存在自己的浏览器 localStorage 里，互相独立，不会共享答案或录音。

## GitHub Pages

把本目录上传到 GitHub 仓库后，在仓库 Settings -> Pages 中选择部署分支，即可在线访问。
