# 新概念英语第一册学习工具

一个离线可用的单页学习工具，用来按课练习新概念英语第一册。

## 功能

- 144 课课程导航
- 奇数课课文精读
- 偶数课隐藏独立课文，只做单词和句型操练
- 单词卡片：英文、音标、词性、中文
- 单词点击发音
- 课文英文朗读：只读英文，并按句子停顿
- 单词测验：英文到中文、中文到英文、音标到英文、词性+中文到英文
- 发音练习：使用浏览器语音识别听你读单词，并给出纠正建议
- 跟读练习：支持任意单词或句子，不限本课单词
- 语法练习：先给例句，再写答案，可自动检查常见错误
- 每个语法练习可单独检查或删除
- 翻译句子训练
- 首次打开设置访问密码，之后输入密码进入
- 本地保存、导入、导出学习数据

## 使用

直接打开 `index.html`。

发音练习需要浏览器支持 Web Speech Recognition。建议使用 Chrome 或 Edge，并允许麦克风权限。

访问密码保存在当前浏览器本地。它适合防止别人随手打开页面，但如果仓库源码公开，前端密码不能当作高强度安全保护。

## GitHub Pages

把本目录上传到 GitHub 仓库后，在仓库 Settings -> Pages 中选择部署分支，即可在线访问。
